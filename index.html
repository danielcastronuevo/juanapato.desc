Cómo implementar
Mismo proceso para todos los productos.
1
Abrí el producto en Tienda Nube → Descripción → botón </> HTML
2
Pegá el código del Ejemplo completo de abajo (o el Snippet base si no usás descripción con características)
3
Editá el texto de descripción y los valores de características. Cambiá data-tabla="1" al número correcto según la tabla de arriba
4
Si el producto está en preventa: eliminá el iframe 🎀 header y descomentá el 🔥 preventa
Snippet base
Solo iframes, sin bloque de descripción. Ideal para productos simples.
